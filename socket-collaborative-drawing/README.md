# Collaborative Drawing with Websockets

1. In terminal, run `npm install`
2. Run `npm start`
3. Open two browser-windows by navigating to `localhost:3000`
4. Start to draw!

![image](example-image.png)


## NPM Dependencies

```
"dependencies": {
  "express": "^4.17.1",
  "socket.io": "^2.3.0"
}
```
